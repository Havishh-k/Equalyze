# RBI Fair Practices Code — Para 3: Non-discrimination

**Regulation:** RBI Fair Practices Code for NBFCs and Banks
**Jurisdiction:** India
**Article:** Para 3
**Domain Tags:** fintech, lending, finance
**Effective Date:** Active

## Text

NBFCs/Banks should not discriminate on grounds of sex, caste, religion, or location in the matter of lending.

In the matter of recovery of loans, the lenders should not resort to undue harassment.

Credit assessment processes, including those driven by artificial intelligence and machine learning models, must be demonstrably free from inherent biases that systematically disadvantage any protected group.

## Compliance Requirements

- Lending algorithms must be audited for bias across gender, caste, religion, and location
- Geographic location (like PIN codes) must not be used as a discriminatory proxy
- Credit assessment processes must be transparent and fair

## Violation Indicators

- Disparate loan approval rates across genders or locations
- Denial of credit based on PIN code (redlining)
- Counterfactual evidence proving location or gender influenced the decision while controlling for valid financial factors
